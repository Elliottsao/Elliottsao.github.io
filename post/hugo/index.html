How to post a Hugo content

Step 1: To enter into your blog root folder, ie, ‘blog’ in my case:

$ cd hugo

$ hugo

Step 2: create a new post(markdown format title.md) by

$ hugo new post/title.md

And then please your familiar text editor to complete your content writing and save. My favourite editor for GitHub writing is Atom IDE and Vim command line.

Local Preview before pushing to GitHub:

$ hugo server -t m10c --buildDrafts

Step 3: Push your content to GitHub repo by running:

$ hugo --theme=m10c --baseUrl="https://elliottsao.github.io/" --buildDrafts

Step 4: Enter into public folder then by running:

$ cd public
$ git add .
$ git commit -m"new change name"
$ git push https://github.com/elliottsao/elliottsao.github.io.git master

Traditional Chinese testing - 中文測試：如何發布一篇Hugo

第一步 - 進入博客文件夾

$ cd hugo
$ hugo

第二步 - 新建markdown文章 打開自己熟悉的文本編輯器，完成這篇.md文章，並保存。

$ hugo new post/title.md

第三步 - 將博客發布到遠程倉庫

$ hugo --theme=m10c --baseUrl="https://elliottsao.github.io/" --buildDrafts

第四步 - 將博客push到遠程倉庫

$ cd public
$ git add .
$ git commit -m"new change name"
$ git push https://github.com/elliottsao/elliottsao.github.io.git master

第五步 - 博客啟動 返回hugo初級文件夾，啟動博客

$ cd ../.. $ cd hugo
